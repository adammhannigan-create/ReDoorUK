# How to add a door to the website

You just send an email. That's the whole thing.

## To put a NEW door up

Send an email to: **adammhannigan@gmail.com**

- **Subject:** start it with the words **NEW DOOR**
  (for example: `NEW DOOR - oak timber up and over`)
- **In the message:** just describe the door in your own words. Helpful things to say:
  - What type it is (up-and-over, side-hinged, roller, etc.)
  - The price
  - What condition it's in
  - Rough age
  - The size (width x height)
  - Whether the size can be adjusted
  - What colours it could be painted
- **Photos:** attach a few. Clear daylight photos sell doors best.

You don't need to lay it out neatly. Write it like a text message. It gets
tidied up automatically before it goes on the site.

### Example

> **To:** adammhannigan@gmail.com
> **Subject:** NEW DOOR - blue side hinged pair
>
> Got a lovely pair of wooden side-hinged doors out of a 70s bungalow.
> Repainted them coastal blue. About 7ft by 7ft, can take the height down a
> bit if needed. Good solid condition. £240. Can do a different colour if they
> want before collection. Photos attached.

That's plenty.

## When a door SELLS

Send another email:

- **Subject:** start it with **SOLD**
- **In the message:** say which door, e.g. "the blue side-hinged pair" or "the oak one".

The website will mark it as sold with a stamp on the photo.

## That's it

No website to log into, no app to learn. Email in, website updates.
If anything looks wrong, tell Adam and he'll sort it in a minute.
