# ReDoor — Dad's email checklist

Print this and stick it up. Send one email per door to the ReDoor Gmail.

## For a NEW door
**Subject line:** start with `NEW DOOR` (e.g. `NEW DOOR - white sectional`)

In the email, include as many of these as you can. Plain English is fine — no need for neat sentences:

- [ ] **Type** — sectional / roller / up and over / side hinged (or "motor" if it's just a motor/opener)
- [ ] **Price** — £ figure
- [ ] **Grade** — A+, A, B, C or D (after you've finished it — this becomes the coloured badge)
- [ ] **Condition** — a few words (e.g. "fully serviced, very good")
- [ ] **Age** — rough is fine (e.g. "about 15 years", "1990s")
- [ ] **Size** — width x height, in whatever units you like (ft, cm, mm) — just say which
- [ ] **Size adjustable?** — yes / no, and by how much if relevant
- [ ] **Colour options** — any RAL colours you can spray it in
- [ ] **A line about it** — anything that makes it stand out (Hörmann motor, two remotes, block pattern, etc.)
- [ ] **Photos** — attach them. More than one is great; the site makes a swipe gallery automatically. Best photo first.

## When a door SELLS
**Subject line:** start with `SOLD`
- [ ] Say which door (the reference like `d-20260707-white-sectional`, or just describe it — "the white sectional one")

That's it. The site updates itself after that. Anything you leave out just gets left off the post — nothing breaks.

---

### Quick reminder on grades (what to write)
- **A+** = repainted, like new
- **A** = excellent, very clean
- **B** = very good, minor marks
- **C** = good, some visible wear
- **D** = functional, shows its age (workshop/outbuilding)
